## ✨ Features

- 🔍 View detailed article content
- 🗑️ Delete articles with confirmation (SweetAlert2)
- 📝 Edit existing articles
- 🔗 Clean navigation using React Router
- 💡 Semantic & responsive UI with Bootstrap
